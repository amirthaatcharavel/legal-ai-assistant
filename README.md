# ⚖️ AI Legal Research Engine using RAG & MCP

An advanced AI-powered legal system that enables intelligent case analysis using Retrieval-Augmented Generation (RAG) and Multi-Context Processing (MCP).

## 🚀 Features

- 📄 Case Summarization
- ⚖️ Case Comparison (Indian Kanoon integration)
- 📂 Document Upload & Analysis
- 🔍 Intelligent Case Retrieval
- 🧠 Context-Aware Legal Reasoning

## 🛠 Tech Stack

- Backend: FastAPI
- Frontend: Vue.js
- LLM: Groq (LLaMA 3)
- Scraping: Selenium + BeautifulSoup
- Database: Firebase
- AI Architecture: RAG + MCP

## 💡 Highlights

- Extracts real judgments from Indian Kanoon
- Compares multiple legal cases intelligently
- Supports uploaded document + case comparison
- Structured legal outputs (Facts, Issue, Judgment, Reasoning)

## 📌 Use Cases

- Legal research automation
- Case law comparison
- Academic legal analysis
- Lawyer productivity tools
